## Setup Checklist

- [ ] Use Chrome (latest version)
- [ ] Disable all extensions

- **Windows:** OBS Studio, Loom, or Windows Game Bar

## Screen Recordings for Help Videos

### Video 1: Platform Overview (3-4 min)
```
Recording Flow:
1. Start at landing page (5 sec)
2. Show login → Dashboard (10 sec)
3. Click through each nav item briefly:
   - Career Goal → show canvas (15 sec)
   - 12 Weeks Plan → show plan (15 sec)
   - Resume Analysis → show score (15 sec)
   - Job Board → show search (15 sec)
   - Job Hunt CRM → show cards (15 sec)
   - Mentoring → show list (15 sec)
4. Return to dashboard (5 sec)
5. Show mobile view (resize browser) (10 sec)
```

### Video 2: Getting Started (3 min)
```
Recording Flow:
1. Start at careerplaybook.app (5 sec)
2. Click "Get Started" button (5 sec)
3. Fill registration form (20 sec)
4. Show email verification screen (10 sec)
5. Complete Step 1: Personal Info (30 sec)
6. Complete Step 2: Career Background (30 sec)
7. Complete Step 3: Career Goals (30 sec)
8. Show dashboard with completion indicator (10 sec)
9. Click to Settings → Profile (10 sec)
```

### Video 3: Career Canvas (4 min)
```
Recording Flow:
1. Navigate to Career Goal (5 sec)
2. If empty, show "Create Canvas" (10 sec)
3. Click into Section 1, type sample text (20 sec)
4. Show auto-save indicator (5 sec)
5. Fill Section 2-3 briefly (30 sec)
6. Show progress percentage updating (10 sec)
7. Scroll through all 9 sections (20 sec)
8. Click "Business View" toggle (10 sec)
9. Show formatted output (15 sec)
10. If multiple canvases, show selector (10 sec)
```

### Video 4: 90-Day Plan (4 min)
```
Recording Flow:
1. Navigate to 12 Weeks Plan (5 sec)
2. Click "Create New Plan" (10 sec)
3. Enter title, set start date (15 sec)
4. Add Week 1 milestone manually (20 sec)

7. Click status dropdown, change to "In Progress" (10 sec)
8. Click status again, mark "Completed" (10 sec)
9. Show progress chart updating (10 sec)
10. Click "Generate with AI" if available (20 sec)
11. Show generated milestones (15 sec)
```

## Image Optimization

### For Pitch Deck
- Format: PNG or JPG
- Resolution: 1920x1080
- File size: <500KB each
- Crop to focus on key UI elements
- Add subtle drop shadow for depth

### For Help Videos
- Resolution: 1920x1080 (1080p)
- Frame rate: 30fps
- Format: MP4 (H.264)
- Cursor: Visible, smooth movements
- Clicks: Deliberate, pause after each
