## Quick Conversion Options

### Option 1: Canva (Recommended for Design)
1. Go to canva.com
2. Search "Pitch Deck" templates
3. Choose a clean, professional template
4. Copy content from INVESTOR_PITCH_DECK.md
5. Add screenshots from SCREENSHOT_GUIDE.md
6. Export as PDF or PPTX

### Option 2: Google Slides
1. Go to slides.google.com
2. Start with blank or choose template
3. Create 15 slides matching the structure
4. Import images via Insert → Image
5. Share link or download as PPTX

### Option 3: PowerPoint
1. Open PowerPoint
2. Use "Pitch Deck" template from File → New
3. Create slides matching structure
4. Add content and images
5. Save as .pptx and .pdf

### Option 4: Pitch.com (Startup Focused)
1. Go to pitch.com
2. Use their investor deck templates
3. Built-in collaboration features
4. Professional export options

## Slide-by-Slide Design Guide

### Slide 1: Title Slide
**Layout:** Centered logo + title
**Elements:**
- Career Playbook logo (large, centered)
- "The AI-Powered Career Coaching Platform"
- Tagline: "Plan. Execute. Land Your Dream Job."
- Founder: Ragid Kader
- Contact: ragid@live.com
- Website: careerplaybook.app
- "Seeking: $500K Seed Round"

**Design Tips:**
- Dark background with light text OR
- Light background with brand colors
- Logo should be prominent
- Clean, minimal design

---

### Slide 2: The Problem
**Layout:** Stats on left, pain points on right
**Elements:**
- Large stat: "75% of resumes never reach humans"
- 4-5 pain point bullets
- Optional: frustrated job seeker illustration

**Design Tips:**
- Use red/orange for problem emphasis
- Bold the key statistics
- Icons for each pain point

---

### Slide 3: The Solution
**Layout:** Feature grid or table
**Elements:**
- "Career Playbook: Your AI Career Coach"
- 6-feature grid with icons:
  - Career Canvas
  - 90-Day Plans
  - AI Resume Analysis
  - Job Tracking CRM
  - Resume vs JD Match
  - Mentorship Hub
- Brief tagline at bottom

**Design Tips:**
- Use brand colors
- Icon for each feature
- Equal spacing in grid

---

### Slide 4: Product Demo
**Layout:** Screenshot showcase
**Elements:**
- 4-6 product screenshots
- Brief caption under each
- "See Career Playbook in Action"

**Design Tips:**
- Screenshots with device mockup frames
- Subtle shadow/depth
- Arranged in 2x3 or 3x2 grid
- Make them clickable in presentation

---

### Slide 5: How It Works
**Layout:** Vertical flow diagram
**Elements:**
- 5 steps with icons:
  1. DISCOVER (Career Canvas)
  2. PLAN (90-Day Plan)
  3. OPTIMIZE (Resume)
  4. EXECUTE (Job Search)
  5. COLLABORATE (Mentorship)
- Arrow flow between steps
- "Dream Job Offer" at bottom

**Design Tips:**
- Use arrows or connector lines
- Consistent icon style
- Step numbers prominent

## Export Settings

### For Presenting
- Format: PPTX (PowerPoint)
- Aspect ratio: 16:9
- Embed fonts: Yes
- Include speaker notes

### For Sharing
- Format: PDF
- Quality: High
- Include links: Yes
- Compress images: Medium

### For Email
- Format: PDF
- File size: <10MB
- Single file
- Password protect (optional)

---

## Presentation Tips

### Before the Meeting
1. Test on presenting device
2. Have backup PDF ready
3. Prepare for Q&A (use Appendix)
4. Know your numbers cold

### During Presentation
1. Spend 1-2 min per slide max
2. Tell stories, not just facts
3. Show product demo live if possible
4. Save Q&A for end

### Key Slides to Nail
1. **Problem** - Make them feel the pain
2. **Product Demo** - Show, don't tell
3. **Traction** - Prove momentum
4. **Ask** - Be clear and confident
